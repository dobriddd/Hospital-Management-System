# Hospital Management System

Software Requirements Specification for a Hospital Management System developed as a university project at Sofia University "St. Kliment Ohridski".

## Authors
- Dobromir Denev
- Nikola Neychev
- Plamena Koleva
- Gergana Dimitrova

## Contents
- Functional and non-functional requirements
- Use-case, Activity, Sequence and Class diagrams
- UI prototypes
- Hardware and software interface specifications

## About
HMS is a software platform designed to automate and streamline the operations of medical institutions. The system covers patient management, electronic health records, appointment scheduling, pharmacy, laboratory management, financial operations, medical transport and more.
